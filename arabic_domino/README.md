# لعبة الدومينو 🁫

لعبة دومينو واقعية بالعربية لـ Android، مبنية بـ Flutter. العب ضد الكمبيوتر بثلاث مستويات صعوبة مع لوحة شرف محلية، مؤثرات صوتية، اهتزاز، ومظهر فاتح/داكن.

## المميزات

- **لعبة كلاسيكية 1 ضد 1** بمجموعة Double-Six (28 حجر)
- **3 مستويات للذكاء الاصطناعي**:
  - **سهل**: حركات عشوائية
  - **متوسط**: ذكاء يفضّل الأحجار الثقيلة والأزواج
  - **صعب**: نظرة استباقية لخطوة + نمذجة حركات الخصم (يتتبع الأرقام التي يفتقدها)
- **واجهة عربية كاملة** بدعم RTL
- **مظهر فاتح وداكن** مع تبديل من الإعدادات (يحفظ الاختيار)
- **مؤثرات صوتية** (وضع الحجر، السحب، الفوز، الخسارة) قابلة للإيقاف
- **اهتزاز Haptic** عند التفاعلات الرئيسية
- **لوحة شرف محلية** تخزن أعلى 20 نتيجة مع المستوى والتاريخ
- **رسم واقعي للأحجار** عبر `CustomPainter` مع ظل، حواف ومنحنيات، ومحاكاة لون عاج
- يدعم الأندرويد مباشرة (تم بناء APK جاهز)

## بنية المشروع

```
lib/
├── main.dart                       # نقطة البداية وإعداد التيمز
├── theme/app_theme.dart            # الثيم الفاتح والداكن
├── models/
│   ├── domino_tile.dart            # حجر الدومينو + المجموعة الكاملة
│   └── game_state.dart             # حالة اللعبة المتغيرة
├── game/
│   ├── game_engine.dart            # القواعد: توزيع، لعب، سحب، تخطي، نهاية الجولة
│   └── ai_player.dart              # خوارزمية الكمبيوتر (3 مستويات)
├── services/
│   ├── preferences_service.dart    # SharedPreferences (إعدادات + لوحة الشرف)
│   └── sfx_service.dart            # المؤثرات الصوتية والاهتزاز
├── widgets/
│   └── domino_tile_widget.dart     # رسم الحجر بـ CustomPainter
└── screens/
    ├── home_screen.dart            # القائمة الرئيسية
    ├── game_screen.dart            # شاشة اللعبة
    ├── settings_screen.dart        # الإعدادات
    └── hall_of_fame_screen.dart    # لوحة الشرف
```

## التشغيل

```bash
flutter pub get
flutter run                # تشغيل على جهاز/محاكي متصل
flutter test               # تشغيل الاختبارات
flutter build apk --release # بناء APK
```

## القواعد المنفّذة

- يوزَّع لكل لاعب 7 أحجار، ويبقى 14 في "البنك" (boneyard).
- يبدأ اللاعب الذي يملك أعلى زوج (مزدوج) باللعب به تلقائياً.
- إذا لم يكن لديك حجر يطابق طرفي السلسلة، تسحب من البنك حتى تجد حجراً صالحاً.
- إذا فرغ البنك ولم يوجد حجر صالح، تتخطى دورك.
- إذا تخطى اللاعبان دوريهما متتاليتين (انسداد)، فالفائز هو من يملك أقل مجموع نقاط.
- في نهاية كل جولة: يحصل الفائز على مجموع نقاط أحجار الخصم المتبقية.
